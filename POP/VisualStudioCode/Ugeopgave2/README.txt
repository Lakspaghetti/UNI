In order to translate the .tex file open your terminal, navigate to the folder containing the .tex file and 
type "pdflatex opgave2.tex" into the terminal. Now the .tex file should be translated and a pdf should've been made, in case 
of any errors with the pdf, try and run the pdflatex command again.

in order to compile and translate the .fsx file navigate to the folder containing it. Now type "fsharpc 2i1.fsx" into the 
terminal and then the 2i1.fsx should be translated and compiled and two new files should appear: a .dll file and a .exe file.
Now in order to launch the code type "mono 2i1.exe" and you are done.
